### 1. 产品基本信息

- **产品名称**：轻量化旅行规划可视化网站

- **产品定位**：本地部署、极简易用的旅行行程规划工具，主打行程列表展示+地点地图联动可视化，满足个人短途/长途旅行的行程编排、点位查看需求

- **目标用户**：个人旅行爱好者、出行规划者、自助游用户

- **核心价值**：快速编排旅行日程，直观查看行程地点分布，本地运行数据私密，操作轻量化无冗余

### 2. 选定技术栈

**前后端分离架构**，贴合个人熟悉的技术体系，无额外学习成本：

- **后端**：SpringBoot 3.2.x + Maven + H2嵌入式数据库 + Spring Data JPA + Lombok

- **前端**：Vue3 + Vite + Element Plus + Axios + Leaflet + OpenStreetMap

- **部署运行**：本地启动运行，数据本地存储，无需云端部署和外置数据库

### 3. 完整功能规划

#### 3.1 旅行计划管理

- 新建旅行计划，填写计划标题、出行开始日期、结束日期

- 查看已有旅行计划列表

- 编辑、删除已有旅行计划

#### 3.2 每日行程规划

- 针对选定的旅行计划，按天添加行程明细

- 行程字段：具体时段、行程地点

- 编辑、删除单条每日行程

- 列表展示对应计划下的所有每日行程

#### 3.3 地图可视化联动

- 右侧地图区域，自动加载对应旅行计划下所有行程地点

- 地点点位标记展示，支持地图缩放、拖拽查看

- 行程与地图点位联动，直观对应行程和地理位置

#### 3.4 基础页面布局

- 整体左右分栏布局，左侧固定为行程规划区，右侧为地图展示区

- 页面适配PC端浏览器，布局铺满屏幕，操作简洁

### 4. 非功能需求

- 本地启动速度快，后端、前端单独启动耗时均不超过3秒

- 接口响应流畅，数据增删改查无明显延迟

- 地图加载稳定，地点解析、点位渲染快速

- 兼容Chrome、Edge等主流PC端浏览器

- 数据本地化存储，无外网数据同步，保障隐私
